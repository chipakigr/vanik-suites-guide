# Vanik Suites Guest Guide

A mobile-first React/Vite digital guest guide for Vanik Suites.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Recommended: Vercel or Netlify.

### Vercel
1. Create account/login at Vercel.
2. Add New Project.
3. Upload/import this project.
4. Framework: Vite.
5. Build command: `npm run build`
6. Output directory: `dist`

Suggested domain:
- `guide.vanikliving.com`
- `stay.vanikliving.com`
